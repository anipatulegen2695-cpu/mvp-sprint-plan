# MVP Sprint Plan (1 апта)

## Спринт мақсаты:
MVP-ның негізгі функцияларын іске қосу

| Спринт тапсырмасы | Күрделілігі | Жауапты тұлға | Дайындық критерийі |
|------------------|------------|---------------|--------------------|
| Жоба құрылымын жасау | Оңай | Developer | Репозиторий құрылды, файлдар бар |
| Негізгі бет (UI) жасау | Орташа | Frontend | Бет ашылады, дизайн көрінеді |
| Деректер базасын қосу | Қиын | Backend | DB қосылған, байланыс жұмыс істейді |
| Авторизация (login) | Орташа | Backend | Қолданушы кіре алады |
| Git арқылы commit жасау | Оңай | Барлығы | Commit тарихы бар |

## Definition of Done:
- Код GitHub-қа жүктелді
- Негізгі функциялар жұмыс істейді
- Қате жоқ немесе минималды
